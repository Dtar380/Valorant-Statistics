API

Esta API se ha hecho publica para que todos puedan acceder a los datos recolectados. La API es de uso libre y no necesita de claves para ser usada.

La API es la misma que se utiliza en esta pagina web, asi que puedes obtener los datos de la misma manera en la que se disponen en la pagina web.

STATICS

La ruta "statics" se utiliza para obtener los datos estaticos de la API. Estos son datos no relacionales, es decir, datos de columnas especificas.

Encabezados

Content-Type string

Parametros de ruta

column string
El parametro "column" corresponde a una de las 5 columnas existentes. Estas son:
  • AGE
  • OCCUPATION
  • GENDER
  • PEAK
  • ROW VALUE
return_type string
El parametro "return_type" corresponde a la forma de devolver los datos. Estas son:
  • table: devuelve la tabla al completo
  • graph: devuelve la tabla preparada para graficar
  • mode: devuelve el valor mas frecuente

Ejemplos de código para "Static"

Ejemplos de solicitud

"/statics/{column}/{return_type}"

QUERY

La ruta "query" se utiliza para hacer consultas a la API. Las consultas devuelven datos relacionales siempre debido a como funcionan.

Encabezados

Content-Type string

Parametros de ruta

query_type string
El parametro "query_type" corresponde al tipo de consulta que se va a realizar. Estas son:
  • Combination: Combinar una cantidad de datos
  • Comparison: Comparar dos columnas acorde a un dato
return_type string
El parametro "return_type" corresponde a la forma de devolver los datos. Estas son:
  • table: devuelve la tabla al completo
  • graph: devuelve la tabla preparada para graficar

Parametros de carga

column string
El valor "column" del payload corresponde al nombre de la columna que se quiere consultar. Estas son:
  • AGE
  • OCCUPATION
  • GENDER
  • PEAK
value string
El valor "value" del payload corresponde al valor que se quiere consultar.

Ejemplos de solicitud
